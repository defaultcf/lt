\begin{frame}{今日に於ける一般的なパスワード管理方法}
  全てのパスワードは，英数字+記号でランダム生成16文字くらい．
  \begin{itemize}
    \item ブラウザのパスワード保存機能を使う
      \begin{itemize}
        \item Webサイトの保存ならこれで十分
        \item サーバのログインパスワードはできない
      \end{itemize}
    \item パスワード管理サービスを使う
      \begin{itemize}
        \item 1Password
        \item LastPass
        \item etc...
      \end{itemize}
  \end{itemize}
\end{frame}
